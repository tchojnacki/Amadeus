# Amadeus - Discord Bot 🤖
General purpose Discord bot based on CQRS and built with Discord.Net.

## Features 🚀
- **BattleRoyale Module**
  - `/br <players>` - play a game of Battle Royale
- **Purge Module**
  - `/purge <count>` - remove recent messages
- **Statistics Module**
  - `/statistics` - show bot's statistics

More modules underway 🚧👷
